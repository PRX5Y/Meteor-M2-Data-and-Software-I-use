Banner mode

    Banner mode can be activated by keyboard shortcut Ctrl+Shift+Enter or Ctrl+Shift+Space. When active, logo/advertisment banner appears at the bottom of Orbitron window. Configuration file, with settings and banner images list, is Config\Banner.set. Its a plain text (ASCII) file and you can edit it in Windows Notepad/Wordpad/Write. File format description is included in it.
    When Config\Banner.set file is not found, Orbitron loads Config\Banner.bmp file as a default banner image. Please remember that your banners must be saved in BMP format. Some example Orbitron/events banners will be available to download on Orbitron website.
