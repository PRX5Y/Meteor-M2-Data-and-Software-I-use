Tips & tricks

All things described in this section are EXPERIMENTAL.

Free name for ISS
You can edit name for ISS (NORAD# 25544). To do this, please add [Tips] section to SETUP.CFG file in Orbitron\Config\ directory, then put a line containing: NameFor25544=XXX under it. XXX is the ISS name you want. The default name is ISS, but if you want to use one from the loaded TLE file, please put these lines to setup.cfg:
    [Tips]
    NameFor25544=

Custom night shadow ratio
Set value of ShadowRatio variable in [Tips] section in SETUP.CFG to customize night shadow ratio. It can be in range from 0 (no shadow) to 10 (absolutely black). Default value is 3.

Colored object's labels in Data panel
Set value of RedNeg variable in [Tips] section in SETUP.CFG to let Orbitron drawing satellite/sun/moon labels, in Data panel, in red or green color depending on positive elevation.

Bold lines at viewports
Setting of BoldLines variable in [Tips] section in SETUP.CFG tells Orbitron to draw track/footprint/terminator lines at viewports with bold style. It useful option when you are preparing screen shots for printing.

Secret console
Pressing Ctrl+Shift+~ in Orbitron main window opens 'Secret' console. Please write help command to list available commands.
