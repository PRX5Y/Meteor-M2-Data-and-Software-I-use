FAQ

Frequently Asked Questions.

I saw satellite pass yesterday at 20:00. Which satellite might it be?
You can check it very easily using Orbitron. Please:
  • click 'Load TLE' button and choose VISUAL.TXT file,
  • track all satellites by choosing Track all in satellite list's context menu,
  • switch to Simulation mode pressing Ctrl + End,
  • enter time of interest in EditBox on center of Main Panel,
  • set jump interval to 0.5 minutes using ComboBoxes bellow,
  • choose Radar viewport (Ctrl + 2),
  • scroll the time (Ctrl + -, Ctrl + +).
Look at radar, watching illuminated satellites for their placements and directions. Situation should be clear, if not check also visual magnitude of object by clicking on it and look at Info panel.

I have a slow computer. What can I do to speed up the application?
There are some things, you can do to make Orbitron faster:
  • track only a few or one satellite,
  • unselect active satellite if you don't need it,
  • turn off world map night shadow and/or terminator,
  • don't use 1 second refresh interval.

I want to see TLE data for selected satellite. Is it possible?
Yes, of course. Please choose Sat/Orbit info - bottom panel, then click right mouse button on satellite name. That's all!

Can you put 'Autoselect satellite after AOS' option?
I think its a bad idea (I don't like pseudo-intelligent applications). Orbitron shows AOS notification in footer, you can press Alt+~ to select notification context satellite. OBSOLETE: Orbitron 3.12 introduced "AOS Notification: Make satellite active" option (Setup/Extra)

I dont like to use decimal degrees!
Please, go to the General tab in Setup and choose DMS output of Lon/Lat. You can enter your coordinates in DMS notation as is described here.

After Orbitron 2.0 installation, my old configuration files don't work!
Because of big changes in Orbitron 2.0, old configuration files are no longer compatible. Sorry...

Orbitron Screen Saver doesn't work when Orbitron is running, is it a bug?
No. At first I allow to run one instance of my application, at second it could look bizarre to see "Orbitron on Orbitron", at third any screen saver is turned off when you are running Orbitron (there are some reasons for that).

TLE Updater doesn't work for me since I installed Service Pack 2 for Windows XP, why?
WinXP SP2 can block Internet access for Orbitorn as default. So, please go to the Windows Control Panel, then choose Firewall icon and in "Exceptions" panel you can add Orbitron application. Now Orbitron is allowed to get data from Internet...
