Keyboard

Main
Ctrl + 1..2 Toggle viewports
F1..F8 Toggle bottom-side panels
Shift + F1...F2 Toggle right-side panels
Alt + L Load TLE
Alt + S Toggle active satellite
Alt + F1 Help
Alt + F2 Screen shot (with shift - viewport only)
Alt + F3 Minimize application
Alt + F4 Exit
Alt + F5 Setup
Alt + F6 Load profile
Alt + F7 Save profile
Alt + F8 Message Board
Alt + Enter Toggle full screen / windowed mode
Ctrl + Enter Maximize viewport
Ctrl + Shift + Enter Banner mode (without right panels)
Ctrl + Space Hide bottom panels
Ctrl + Shift + Space Banner mode (with right panels)
Time manipulation
Ctrl + Home Real-time mode
Ctrl + End Simulation mode
Ctrl + ~ Refresh/Jump interval status
Ctrl + Z/X Set refresh interval
Ctrl + Q/W/A/S Set jump interval
Ctrl + - Jump backward
Ctrl + + Jump forward
Ctrl + PgUp Auto backward simulation
Ctrl + PgDn Auto forward simulation
Others
ESC Remove special message from footer or open QuickMenu if none
Shift + ESC Recall last red-color message
Alt + ~ Go to red-color message' context satellite
F9 Open QuickMenu
F10 Sat on track: begin/middle/end
F11 Time Panel: On/Off
F12 Mini Radar: On/Off/Auto
In screen saver: turns to standard application mode (experimental)
