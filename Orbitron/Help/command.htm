Command-line parameters

Parameters list
RunDriver=xxx Run driver "xxx" after startup (xxx is the name from drivers list)
Force Allows new instance of Orbitron
Silent Orbitron will start and close itself without any dialog boxes
IgnoreCRC If used CRC checking is ignored
Minimized Starts application with minimized window
Example
ORBITRON.EXE RunDriver=MyDDE Silent Minimized
ORBITRON.EXE Force
